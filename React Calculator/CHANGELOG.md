# Changelog

## [0.1.0] — 2026-06-21

### Added
- Classic calculator UI with dark theme
- Basic operations: addition, subtraction, multiplication, division
- Clear (C) and backspace functionality
- Keyboard support for number entry
- mathjs expression evaluation
- SonarQube, Qodana, ESLint, Prettier configurations
- AI context files (.cursorrules, copilot-instructions.md, CLAUDE.md)
