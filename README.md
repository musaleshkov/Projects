# 🚀 Projects

A collection of projects spanning various technologies — React, TypeScript, Next.js, blockchain, and more.

---

## 📂 Projects

### 🏥 [Health Landing Page](./Landing%20Page/)
A modern, animated health assessment landing page with an interactive multi-step quiz, glassmorphism design, and internationalization (EN/FR).

**Stack:** Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Redux Toolkit · i18next

### 🧮 [React Calculator](./React%20Calculator/)
A dark-themed, glassmorphism calculator with keyboard support and safe expression evaluation via mathjs.

**Stack:** React 19 · JavaScript · mathjs · CSS Custom Properties

### 💰 [Crypto Wallet App](./Crypto%20Wallets/crypto-wallet-app/)
A crypto wallet dashboard with MetaMask integration, multi-chain token balances, and live CoinGecko market data.

**Stack:** React 19 · TypeScript · ethers.js v6 · CoinGecko API

## 🚦 Running a Project

Each project has its own dependencies and scripts. Navigate into the project directory and follow its README:

```bash
cd "Landing Page"        # or "React Calculator" or "Crypto Wallets/crypto-wallet-app"
npm install
npm start
```
