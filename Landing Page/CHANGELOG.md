# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2025-01-20

### Added
- Next.js 15 with Turbopack dev server
- React 19 with TypeScript 5.7
- Interactive quiz flow with Redux Toolkit state management
- React Query for server state and API caching
- Tailwind CSS with forms and typography plugins
- i18next internationalization (English + French)
- Cypress E2E and component testing with cypress-axe
- Sentry error tracking and LogRocket session replay
- Responsive landing page (Hero, Info, Footer sections)
- SonarQube, Qodana, ESLint (flat config), Prettier configurations
- AI context files (.cursorrules, CLAUDE.md)
- .editorconfig, .nvmrc for consistent development environment

### Changed
- Updated Node.js engine requirement to >=22.11.0
- Updated npm requirement to >=11.0.0
- Migrated to ESLint flat config (v9)
- Consolidated PostCSS config to ESM format
